# 👕 DressUp – Character Customization Mini Game

A simple browser-based mini game where you can dress up a character by cycling through different clothing options.

## 🚀 About the Project

**DressUp** is a front-end mini project built with **HTML, CSS, and JavaScript**, focused on practicing DOM manipulation and basic game logic.

The player can customize a character by selecting different items such as:

* Hats 🧢
* Shirts 👕
* Pants 👖
* Shoes 👟

Each category cycles through multiple options using interactive buttons.

---

## 🎮 Features

* Click-based outfit selection
* Dynamic image updates using JavaScript
* Layered character rendering (body + clothes)
* Simple and intuitive UI

---

## 🛠️ Technologies Used
